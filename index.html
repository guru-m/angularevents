Keyboard Events

ng-keydown, ng-keypress, ng-keyup

Keyboard Events

They are basically triggered when the keys are pressed or released

  • ng-keydown

    is trigged when the key is pressed and it is on the down position

  • ng-keypress

    is triggered when the key is pressed. It does not recognise CTRL, ALT nor SHIFT

  • ng-keyup

    is triggered when the key is release

Mouse Events

ng-mousedown, ng-mouseup, ng-mouseenter, ng-mouseleave, ng-mousemove, ng-mouseover

Mouse Events

They are triggered when the mouse moves or the buttons are clicked.

  • ng-mousedown

    is trigged when the mouse key is pressed and it is on the down position
    The mouse click it will change "keyword

    ng-mousedown

  • ng-mouseup

    is triggered when the mouse key is release

  • ng-mouseenter

    is triggered when the mouse pointer enters the area of the object which is listening to the event

  • ng-mouseleave

    is triggered when the mouse pointer leaves the area of the object which is listening to the event

  • ng-mousemove

    is triggered everytime that the mouse moves on the area where the object is listening to the event

  • ng-mouseover

    is triggered everytime that the mouse moves is over the area where the object is listening to the event

We have to initialize the mouse object with in the scope inside the script values as 0 then we can use the keywords
